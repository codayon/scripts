## Quick Run

```bash
bash <(curl -s https://raw.githubusercontent.com/codayon/vite-react.js/main/one.sh)
```

## Run Locally

```bash
git clone https://github.com/codayon/vite-react.js.git
cd vite-react.js
chmod +x one.sh
./one.sh
```

> **NOTE**: Your shell needs to be Bash (e.g., Git Bash or WSL on Windows).
